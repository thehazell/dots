## Symlinking
If you prefer to keep this in your home directory to work on (which is my preference), you can symlink `shell.json` to the proper location with the following command:
```shell
ln -s ~/Projects/dots/caelestia/shell.json ~/.config/caelestia/shell.json
```

You may have to create `~/.config/caelestia/`, as it usually doesnt not come pre-created with a caelestia install.

## Move the file
You can also just move this file to `~/.config/caelestia/shell.json`, if you want. Make sure the caelestia directory exists, as described above.
